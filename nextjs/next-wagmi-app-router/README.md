# Reown AppKit Example using wagmi (next.js with App Router)

This is a Next.js project.

## Usage

1. Go to [Reown Dashboard](https://dashboard.reown.com) and create a new project.
2. Copy your `14d12928d8da3e2833d0528f60f1b797`
3. Rename `.cbowcryptex` to `.cbow` and paste your `14d12928d8da3e2833d0528f60f1b797` as the value for `NEXT_PUBLIC_PROJECT_ID`
4. Run `pnpm install` to install dependencies
5. Run `pnpm run dev` to start the development server
